Instructions for Claude — READ FIRST

Working method: Claude delivers complete files named with the convention in Section 09. Aurora applies Cmd+A → Delete → Paste → Cmd+S in the correct VS Code tab. Always verify breadcrumb path before pasting. Deploy: cd ~/kantier && git add . && git commit -m "..." && git push

What is Kantier?

Kantier is Albania's first construction site management SaaS, built by Aurora Baba (architect, UCL London, no coding background). Stack: Next.js 16.2 + Supabase + Vercel. Claude acts as a PMP-accredited construction PM familiar with Procore, Fieldwire, PlanRadar. PMBOK and ISO 9001 standards apply to all document decisions.

Critical Rules for Claude

  • Never call loadData() after field saves — causes race condition where task dates snap back. Update local state with setSelectedTask + setDetyrat instead.
  • Date inputs must save on onChange, not onBlur — onBlur races with loadData() re-fetch.
  • File naming convention is mandatory — see Section 09. Always use route_path_page.tsx format.
  • params.id must be typed — always const params = useParams() then const projektId = params.id as string on the next line. Never let sed replace params.id inside its own declaration.
  • No sed on TypeScript files — use Python string replace or str_replace tool instead.
  • Upload files one at a time — multiple uploads overwrite each other. Wait for line count confirmation before sending the next file.
  • PDF only, no Word export — Word is editable and defeats ISO 9001 document control.
  • data_mbarimit hidden from UI — field preserved in DB but never shown.

Collaboration Architecture — @mentions + notifications

Required new Supabase table

CREATE TABLE notifikime (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  projekti_id uuid REFERENCES projektet(id),
  perdoruesi_email text NOT NULL,     -- recipient
  lloji text NOT NULL,                -- 'mention' | 'task_assigned' | 'rfi_answer'
                                      -- | 'doc_issued' | 'inspection_due' | 'weekly_report'
  referenca_id uuid,                  -- task id or document id
  referenca_tipi text,                -- 'detyre' | 'dokument'
  teksti text,                        -- notification message
  lexuar boolean DEFAULT false,
  created_at timestamptz DEFAULT now()
);

@mention flow (in task comments and documents)

  1. User types @ in a comment field or document text area
  2. Dropdown appears showing anetaret of the current project (from anetaret table)
  3. Selecting a person inserts @email as a styled badge in the text
  4. On save: insert row to notifikime + trigger Supabase Edge Function → Resend email to recipient
  5. Recipient email contains: project name, what was said, a direct link to the task or document
  6. Unread count shown as badge on the bell icon in the nav

@document reference flow

  1. User types @RFI or @COR in a comment or description
  2. Dropdown shows matching document numbers from dokumentet table (same project)
  3. Selecting inserts a clickable badge: RFI-2026-PT01-001
  4. Stored as a reference in doc_references jsonb field
  5. Renders as a tappable chip that navigates to the document detail

Task assignment notification

When caktuar_email is changed on a task: insert notifikime row + send email with task title, deadline, and direct link. This requires caktuar_email to become a dropdown from anetaret (P2 bug fix must come first).

The ROI argument

One prevented rework event saves €2,000–€20,000 on a typical residential project. One missed inspection that stops work costs €500–€2,000 per day. One documented COR that survives a client dispute is worth the annual subscription many times over. The pitch writes itself.

The Albanian market reality

€49/month is affordable for any contractor billing over €200,000/year. The resistance is not price — it is habit change. The strategy is to make the free tier genuinely useful so contractors adopt it on small projects, then upgrade when they win a larger contract. Daily reports and client-facing PDF exports are the features that justify the upgrade.

International expansion play

For contractors in Germany, Austria, Switzerland (large Albanian diaspora contractor communities), €49 is negligible. The expansion is not more features — it is German and Italian language localisation. A single language addition opens a €10B+ addressable market.

Shared ISO 9001 header — all types

Every document shares: Document Number, Type, Revision, Status (Draft/Issued), Prepared by, Approved by, Date Issued. Auto-generated. Cannot be edited after issuance.

TypeBodyEditable untilKey action

CORReason + scope + line-item cost table + financial summary + schedule impact + approvalApproval editable until approved/rejected—
DCOBefore/after panels (red/green) — orig + revised drawings + specs + screenshots + affected docsUntil issued—

Document numbering

Format: TYPE-YEAR-CODE-NNN — e.g. RFI-2026-PT01-001
Code = kodi_projektit from projektet table (max 8 chars). Each type has its own sequence. Revisions R00–R05. Re-issuing = new revision. New document = new number.
