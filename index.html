Instructions for Claude — Read First

Working method: Claude delivers complete files named with the convention in Section 09. Aurora applies Cmd+A → Delete → Paste → Cmd+S in the correct VS Code tab. Always verify the breadcrumb path before pasting. Deploy: cd ~/kantier && git add . && git commit -m "..." && git push

What is Kantier?

Albania's first construction site management SaaS built by Aurora Baba (architect, UCL London, no coding background). Stack: Next.js 16.2 + Supabase + Vercel. Claude acts as a PMP-accredited construction PM familiar with Procore, Fieldwire, PlanRadar. PMBOK and ISO 9001 standards apply to all document decisions.

Critical Rules — Never Break These

RuleReason
Never call loadData() after field savesRace condition — dates snap back to DB value. Update local state with setSelectedTask + setDetyrat instead.
Date inputs save on onChange, not onBluronBlur races with loadData() re-fetch. onChange is immediate.
File naming convention is mandatorySee Section 09. All files are named page.tsx in VS Code — the route-based name prevents wrong-tab mistakes.
params.id must be typedLine 1: const params = useParams() then line 2: const projektId = params.id as string. Never inline.
No sed on TypeScript filessed breaks on special chars. Use Python string replace or str_replace tool.
Upload files one at a timeMultiple uploads overwrite each other. Wait for line count confirmation before next upload.
PDF only, no Word exportWord is editable — defeats ISO 9001 document control.
data_mbarimit hidden from UIField preserved in DB but never shown to users.
Shkarko Draft ≠ Regjistro & Lësho"Shkarko Draft" prints only — no DB save. "Regjistro & Lësho" generates fresh number, saves to DB, sets statusi=issued, then prints. Never conflate the two.
planet/markup stores thumbnailmarkup jsonb field stores {thumbnail: "data:image/jpeg..."}. Always check Array.isArray(plan.markup) before spreading — old rows are [].
foto_urls stores objects, not plain stringsEach entry is {url, email, data}. Parse with: typeof v === 'string' && v.startsWith('{') ? JSON.parse(v).url : (v?.url || v). Never use raw entry as img src.
Sidebar nav: Dashboard + Njerëzit onlyDetyrat/Planet/Raportet are project-specific routes — they 404 as global routes. Sidebar links only to /dashboard and /dashboard/njerezit.

// src/app/api/ai-report/route.ts
export async function POST(req: Request) {
  const { projektId, date, weather } = await req.json()
  const tasks = await supabase.from('detyrat')
    .select('titulli, statusi, kategoria, caktuar_email')
    .eq('projekti_id', projektId).neq('statusi', 'mbaruar')
  const response = await anthropic.messages.create({
    model: 'claude-sonnet-4-6', max_tokens: 1000,
    messages: [{ role: 'user', content:
      `Gjenero raportin ditor për ${date}. Mot: ${weather}.
       Detyrat aktive: ${JSON.stringify(tasks.data)}.
       Shkruaj në shqip. Format: punëtorët, puna e kryer, materialet, probleme, plan nesër.`
    }]
  })
  return Response.json({ report: response.content[0].text })
}

DecisionRationale
Light theme everywhere — white nav, #F4F5F7 bg, #1C2333 inkMatches Fieldwire. No dark mode.
Teardrop pins + category acronyms — color = priorityType + urgency readable at a glance on crowded plans.
Flat dropdowns — white bg, colored dot, no emojiProfessional. Emojis render badly in PDFs.
komentet stores activity + commentsNo extra table. type:'activity' = pencil icon. No type = chat bubble.
TYPE-YEAR-CODE-NNN numberingPMBOK standard. Each type has own series.
PDF only, no Word exportWord defeats ISO 9001 document control.
Revisions R00–R05RPT locked at R00. Others up to R05.
Shkarko Draft ≠ Regjistro & LëshoPrint without committing vs. formal issuance. Procore model.
Sheet # in pershkrimi columnReuses existing planet.pershkrimi — no migration needed.
Thumbnails in markup jsonbGenerated client-side on upload (PDF.js, scale 0.4, JPEG 70%). Old rows are [] — always check Array.isArray before spreading.
devicePixelRatio in pdf-rendererCanvas renders at 2× physical pixels on retina. Matches Fieldwire sharpness.
5-level priority (not 9)Matches Fieldwire. Legacy values kept in color maps for old tasks.
Per-project pricing, not per-userConstruction sites have many workers. Per-user kills adoption.
Sidebar: Dashboard + Njerëzit onlyDetyrat/Planet/Raportet are project-scoped. Global sidebar links only to truly global routes to prevent 404s.
foto_urls as {url,email,data} objectsEnables Foto page metadata (who uploaded, when). Parser handles all 3 legacy formats (string, object, JSON-string).
Gantt uses pixel-per-day (38px)Fixed density, continuous canvas. 1-year span centred on today. Drag-to-pan matches Fieldwire UX.

Document System

Shared ISO 9001 header — all 8 types. Every issued document has: Document Number (fresh sequential), Type, Revision, Status (always ISSUED), Prepared by, Approved by, Date Issued.

Document numbering: Format TYPE-YEAR-CODE-NNN (e.g. NCR-2026-PT03-001). Code = kodi_projektit (max 8 chars). Each type has its own sequence. Number generated fresh at issue time by counting existing docs of that type.
