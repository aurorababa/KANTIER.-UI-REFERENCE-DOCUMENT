Working method: Claude delivers complete files named with the convention in Section 09. Aurora applies Cmd+A → Delete → Paste → Cmd+S in the correct VS Code tab. Always verify the breadcrumb path before pasting. Deploy: cd ~/kantier && git add . && git commit -m "..." && git push

What is Kantier?

Albania's first construction site management SaaS built by Aurora Baba (architect, UCL London, no coding background). Stack: Next.js 16.2 + Supabase + Vercel. Claude acts as a PMP-accredited construction PM familiar with Procore, Fieldwire, PlanRadar. PMBOK and ISO 9001 standards apply to all document decisions.

Critical Rules — Never Break These

RuleReason
Never call loadData() after field savesRace condition — dates snap back to DB value. Update local state with setSelectedTask + setDetyrat instead.
Date inputs save on onChange, not onBluronBlur races with loadData() re-fetch. onChange is immediate.
File naming convention is mandatorySee Section 09. All files are named page.tsx in VS Code — the route-based name prevents wrong-tab mistakes.
params.id must be typedLine 1: const params = useParams() then line 2: const projektId = params.id as string. Never inline.
No sed on TypeScript filessed breaks on special chars. Use Python string replace or str_replace tool.
Upload files one at a timeMultiple uploads overwrite each other. Wait for line count confirmation before next upload.
PDF only, no Word exportWord is editable — defeats ISO 9001 document control.
data_mbarimit hidden from UIField preserved in DB but never shown to users.

AI Strategy — S8 Decision

Decision: Build value now without API keys. Add AI generation as Phase 3 feature. Guides + structured forms are immediate value. AI generation is the Pro upsell.
📖 Writing Guides — Now
✓ Live
Collapsible guides on all 5 doc types. RFI: 3-part rule + template + Albanian example. INS: measurable checklist items. COR: acceptable reasons. DCO: what to include. Zero API cost. Teaches PMs how to write professional documents.
📋 Structured RDT — Now
✓ Live
Daily report as structured form (not free text). Weather dropdown, workers table, materials, problems, next-day plan. Takes 3 minutes on site. Professional PDF sent to client. Creates the daily habit of opening Kantier.
🤖 AI Daily Report — Phase 3
PRO
Next.js API route: /api/ai-report. Anthropic claude-sonnet-4-6. Fetches today's active tasks from Supabase + weather from VisualCrossing (free tier, passes project adresa). Claude generates Albanian narrative. PM edits and registers. 30 min → 3 min.
💬 AI RFI Writer — Phase 3
PRO
PM types informally: "kanali nuk kalon te dhoma 3". Claude reformats as proper RFI: drawing ref, grid location, specific question, required-by date. Same /api/ai-report route. Stored in form_data.rfiQuestion field.
🔍 Semantic Search — Phase 4
Future
pgvector extension on Supabase. Embed all document content + task descriptions. Query: "all RFIs about electrical on first floor". Returns ranked results across tasks, docs, comments.
📸 Photo Tagging — Phase 4
Future

// src/app/api/ai-report/route.ts
export async function POST(req: Request) {
  const { projektId, date, weather } = await req.json()

  // 1. Fetch today's active tasks from Supabase
  const tasks = await supabase.from('detyrat')
    .select('titulli, statusi, kategoria, caktuar_email')
    .eq('projekti_id', projektId)
    .neq('statusi', 'mbaruar')

  // 2. Fetch weather from VisualCrossing (free tier)
  // GET https://weather.visualcrossing.com/VisualCrossingWebServices/rest/services/
  //     timeline/{address}/{date}?key={VISUAL_CROSSING_KEY}&unitGroup=metric

  // 3. Call Claude
  const response = await anthropic.messages.create({
    model: 'claude-sonnet-4-6',
    max_tokens: 1000,

Date inputs save on onChangeonBlur races with loadData(). onChange is immediate.
